NetPeek — Ayuda

La opción 4 del menú de NetPeek muestra información básica sobre puertos de red comunes y para qué servicios suelen usarse.

Un puerto abierto no significa que exista una vulnerabilidad. Solo indica que hay un servicio escuchando conexiones.

Puertos comunes

21 → FTP — transferencia de archivos
22 → SSH — acceso remoto seguro
23 → Telnet — acceso remoto sin cifrado
25 → SMTP — envío de correos
53 → DNS — resolución de dominios
80 → HTTP — páginas web
443 → HTTPS — web cifrada
445 → SMB — archivos en red Windows
3389 → RDP — escritorio remoto Windows
3306 → MySQL — base de datos
5432 → PostgreSQL — base de datos
8080 → HTTP alternativo

¿Qué hace NetPeek?

NetPeek intenta conectarse a un puerto usando TCP sockets. Si la conexión responde, el puerto se marca como abierto.

Esto es una técnica básica de reconocimiento de red.

Uso ético

Usa esta herramienta solo en:

  • tu propia máquina
  • laboratorios de práctica
  • sistemas donde tengas permiso

Escanear redes sin autorización puede ser ilegal.
